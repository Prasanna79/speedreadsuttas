# X Launch Post

I built something a bit unusual: a speed-reader for Buddhist suttas.

I wanted to read more of the Pali Canon, but long screen reading kept turning into tab-hoarding and half-finished suttas. So I built SuttaSpeed to reduce the friction: one-word / chunked RSVP reading, adjustable pace, translation switching, and a cleaner way to stay with the text.

If you want to try it, here’s MN19 in English (Sujato):
https://suttaspeed.com/read/mn19/en/sujato

Would love to know if it actually helps you stay with the sutta.
